## Заголовок второго уровня

Абзац после заголовка. Важно проверить отступы между заголовком и текстом — они должны быть комфортными для чтения. Слишком маленький отступ делает текст тесным, слишком большой — разрывает связь.

### Заголовок третьего уровня

Ещё один абзац. Заголовки третьего уровня обычно используются для подразделов внутри раздела. Проверяем что размер шрифта заметно отличается от второго уровня, но меньше него.

#### Заголовок четвёртого уровня

Абзац под четвёртым уровнем. На некоторых темах четвёртый уровень почти не отличается от обычного текста — это нормально.

## Списки

Абзац перед списком. Важно проверить отступ между текстом и началом списка.

- Первый пункт простого списка
- Второй пункт
- Третий пункт с более длинным текстом, который может переноситься на следующую строку если экран достаточно узкий
  - Вложенный пункт первый
  - Вложенный пункт второй
    - Третий уровень вложенности
- Четвёртый пункт после вложенного

Абзац между списками.

1. Первый пункт нумерованного списка
2. Второй пункт
3. Третий пункт
   1. Вложенный нумерованный
   2. Ещё один вложенный
4. Четвёртый пункт

## Код

Абзац с `inline кодом` внутри текста. Проверяем что шрифт и фон кода заметно отличаются от основного текста.

Блок кода на JavaScript:

```javascript
function hello(name) {
  console.log(`Hello, ${name}!`);
  return name.toUpperCase();
}

hello('world');
```

Блок кода на CSS:

```css
body {
  font-family: sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #333;
}
```
